## Overview

This project is a web-based simulator for mechanical gears. The goal is to provide an intuitive tool that helps users calculate and visualize relationships between gears of different sizes.

Users can enter key parameters such as the number of teeth on the driving gear and driven gear. From those inputs, the application computes values such as gear ratio, speed ratio, and torque relationships.

## Core Experience

- Interactive input for gear parameters.
- Real-time calculations for core mechanical relationships.
- Visualization of two or more meshing gears based on user input.
- Educational support through terminology and concept explanations.

## Target Audience and Purpose

The primary audience is students in introductory engineering courses who want to learn and explore gear design fundamentals.

The overall objective is to provide an educational experience that combines:

- clear numerical output,
- visual understanding of gear interactions,
- and supporting explanations of what each computed value means in design context.
